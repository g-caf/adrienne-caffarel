================================================================================
                        QUICK START
================================================================================

LOCAL DEVELOPMENT
  1. cd /path/to/adrienne-personal-site
  2. npm install
  3. cp .env.example .env
  4. npm run migrate
  5. npm run dev
  6. Visit http://localhost:3000

DEPLOY TO RENDER
  1. Push to GitHub
  2. Create PostgreSQL on Render
  3. Create web service on Render
  4. Set environment variables
  5. Render auto-deploys

See DEPLOYMENT.md and LOCAL_TESTING.md for detailed instructions.
